1. Click on ‘Choose Image’ to select the image to be segmented
2. Two windows pop up. Left window provides the image for reference to the user. The right window is the one where segmentation occurs.
3. Click on ‘Segment’
4. Using the mouse, select seed point(s) on the image. More than one seed point may be required for the image based on its complexity
5. You can click on ‘Close Image’ if you wish to close the current image and work on another image
6. ‘About Us’ section provides details about the project group
7. Click on ‘Exit’ to close the application




PS. - MATLAB runtime compiler(MyAppInstaller_mcr.exe) may be required while using the application - Region_Growing_Segmentation.exe
No need for this if running the code via MATLAB
